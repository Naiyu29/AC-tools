核心功能與操作

  1. 輸入/匯入資料: 可手動貼上資料至下方三個斷面區塊，或點擊「匯入資料」選擇先前儲存的 .json 檔案。
  2. 分析計算: 點擊「處理資料並分析」按鈕，程式將自動計算並顯示下方的所有分析報告。
  3. 儲存/匯出資料: 點擊「儲存結果」可將當前分析命名並存入雲端公開列表。點擊「匯出資料」可將輸入區的資料下載為 .json 檔。
  4. 管理紀錄: 雲端列表中的紀錄任何人皆可讀取、編輯名稱或刪除。
  5. 查看報告: 各分析報告區塊的標題列皆可點擊以展開或收合內容，方便聚焦查閱。
